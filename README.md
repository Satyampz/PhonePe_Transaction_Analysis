# PhonePe_Payments_Analysis
# 📊 PhonePe Transaction Analysis Dashboard

An interactive Power BI dashboard built to analyze PhonePe transaction data across different service types, payment status, customer demographics, and transaction trends.

---

## 📌 Project Overview

This dashboard provides insights into:

- Total Transactions
- Transaction Value
- Unique Users
- Success Rate
- Monthly Transaction Trends
- Service-wise Analysis
- Age Group Contribution
- Weekday vs Weekend Usage
- Top Users by Transaction Value
- Payment Status Analysis

The dashboard helps identify customer behavior, popular services, and transaction performance.

## Insights

- Success rate is approximately 96%.
- Loans contribute the highest transaction value.
- Weekday transactions account for over 71% of total usage.
- Millennials and Gen X contribute the largest user base.
- Credit Score service generates the highest transaction value.
- UPI ID and Self Account have the highest transaction counts.
